## 核心特性

### 🍱 Premium Bento Box 仪表盘 (New!)

新一代仪表盘采用了极致的 **SaaS 级 Bento Box** 设计语言：

- **极致对比排版**：核心数值（价格、倍率等）采用大号加粗字体，配合微缩标签化次要信息，数据层级清晰。
- **全站中文化**：从指标标签到 AI 建议，全面适配中文专业术语。
- **深色/浅色模式**：支持一键切换并自动保存偏好（localStorage）。
- **响应式网格**：基于 CSS Grid 的自适应布局，在手机与桌面端均有完美体验。
- **隐私保护**：采用 `noindex, nofollow` 策略，且不泄露任何 API 密钥或敏感账户余额。

### 📊 多因子量化引擎

倍率 `m` 由百分比权重系统动态计算，而非简单加减分：

| 因子 | 权重占比 | 逻辑 |
|:---|:---|:---|
| MA 偏离度 | 10%–25% | MA20 / MA60 / MA120 / MA250 四级偏离比例 |
| RSI | 15%–20% | 超卖 (<30) 加仓，超买 (>65) 减仓 |
| ADX 趋势过滤 | 门控 | ADX>25 且下行趋势 → 禁止加仓 |
| MACD 门控 | 权重修正 | 动量加速下跌阶段，加仓信号按权重进行衰减 |
| ATR 动态头寸 | 自适应 | 根据波动率水平自动调整单次投入规模 |
| 宏观因子 | 修正 | 美债飙升/美元走强 → 下调非美资产倍率 |

最终倍率约束在 `[0.3, 3.5]` 之间。

### 🛡️ AI 首席风控官 (CRO)

接入智谱 GLM-4-Flash，角色设定为首席风控官：

- **中文决策**：强制 AI 使用专业中文输出简洁、有逻辑的投资风控建议。
- **背离分析**：深度检测 RSI/MACD 等技术指标的背离，判断“多头陷阱 (Bull Trap)”风险。
- **合理性审核**：对量化系统计算出的倍率进行“人类逻辑”二次审核。

### 📈 内联历史回测

每次运行自动对所有资产进行 **180 天策略回测**：

- **胜率 (Win Rate)**：买入信号后一段时间内盈利的概率。
- **回撤风控**：若历史最大回撤 **> 15%**，系统将自动对该资产倍率执行 **×0.8 惩罚因子**。
- **可视化附件**：资金曲线图表随邮件自动分发。

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

| 变量名 | 说明 |
|:---|:---|
| `ZHIPU_API_KEY` | 智谱 AI API Key |
| `EMAIL_USER` | 发信邮箱 (如 QQ 邮箱 SMTP 用户) |
| `EMAIL_PASS` | 邮箱授权码 |
| `EMAIL_RECEIVER` | 报告接收邮箱 |

### 3. 配置资产 (`assets.json`)

```json
{
    "标普500": { "ticker": "^GSPC", "base_amount": 100, "currency": "USD" },
    "纳斯达克": { "ticker": "^IXIC", "base_amount": 50, "currency": "USD" }
}
```

### 4. 运行

```bash
python main.py
```
